# 🛒 Blinkit Sales Dashboard

## 📊 Project Overview
This project presents an interactive dashboard built using Power BI to analyze Blinkit (grocery delivery) sales data.  
It helps in understanding sales performance, product demand, and customer trends.

---

## 🛠️ Tools Used
- Power BI
- Excel

---

## 📌 Key Features
- Sales performance analysis
- Product category insights
- Revenue trends and patterns
- Customer demand analysis
